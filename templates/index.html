Game Overview

Tabletop, procedurally generated role-playing game. End goal will be a physical game with a virtual option. You should receive a download code for the virtual option should you purchase the physical version. The digital version should also be able to be shared with up to whatever the maximum number of players ends up being. The license holder pays for the game, then players are able to play for free, just like any board game or TTRPG would work.
